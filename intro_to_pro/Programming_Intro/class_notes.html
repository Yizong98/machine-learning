Part 1--World Wide Web

Web is composed of four main parts: User, Browser, Internet, and Servers.
Users are the one who use the Internet. Browsers are responsible for displaying files on the web. Internet is a large computer network that contain infinite amount of data. Servers are computers that host files that make up the web/
